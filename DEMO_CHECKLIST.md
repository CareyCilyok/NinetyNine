# NinetyNine Demo Checklist

A repeatable 2-3 minute demo walkthrough.

## Setup
- [ ] Have the app running (from zip or source)
- [ ] Clear any previous game state

## Demo Steps

### 1. Launch & Overview (~30 sec)
- [ ] App launches without crash
- [ ] Dark neon theme displays correctly
- [ ] "Ninety-Nine Pool" header is visible

### 2. Start New Game (~30 sec)
- [ ] Enter player name (optional)
- [ ] Enter venue name (optional)
- [ ] Select table size (optional)
- [ ] Click "New Game"
- [ ] Verify 9 empty frames appear

### 3. Score Frame 1 (~45 sec)
- [ ] Use +/- buttons to set Break Bonus (0 or 1)
- [ ] Use +/- buttons to set Ball Count (0-10)
- [ ] Observe frame score updates (Break + Balls)
- [ ] Click "Complete Frame"
- [ ] Verify running total updates
- [ ] Verify Frame 2 becomes active

### 4. Score Frame 2 (~30 sec)
- [ ] Set Break Bonus and Ball Count
- [ ] Complete the frame
- [ ] Verify total score accumulates correctly

### 5. Optional: Perfect Frame (11 points)
- [ ] Set Break Bonus = 1, Ball Count = 10
- [ ] Observe "PERFECT" indicator and gold styling

### 6. Optional: Perfect Game (99 points)
- [ ] Complete all 9 frames with 11 points each
- [ ] Observe "LEGENDARY! PERFECT 99!" message
- [ ] Confetti animation displays

### 7. End Demo
- [ ] Click "End Game" or complete all frames
- [ ] Final score displayed
- [ ] "Play Again" button works

## Validation
- [ ] All scores align with README rules
- [ ] No crashes or UI glitches
- [ ] Responsive on window resize
