## Processing notes

These images are served directly as committed. For production optimization, consider:

- Converting to WebP with a JPEG fallback (smaller payload, ~30-50% reduction)
- Generating multiple resolution variants via `<picture>` + `srcset` for responsive hero backgrounds
- Stripping remaining EXIF metadata (not strictly necessary since Pexels already strips most)

None of these are strictly required for v1.
